# Squares

Java okenni hra pro dva hrace inspirovana hrou tecky a ctverce.

## Funkce verze 1.0

- hra na jednom PC
- sitova hra v rezimu hostitel/klient
- volitelna velikost pole 5x5 az 10x10
- skore, cas hry a cas premysleni kazdeho hrace
- restart hry a zmena velikosti pole z menu
- volitelne zvuky

## Spusteni

```bat
run.bat
```

## Sestaveni

Projekt je kompatibilni s Java 8.

```powershell
mvn package
```
